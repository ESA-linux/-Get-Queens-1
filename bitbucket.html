Latest updates

We deploy updates to Bitbucket Cloud frequently. As a Bitbucket developer, it's important that you're aware of the changes. The resources below will help you keep track of what's happening.

Recent announcements

Changes announced in the Atlassian Developer blog are usually described in more detail in this documentation. The most recent announcements are documented in detail below:

  • Change notice - Connect app webhooks to be restricted by scopes effective 15 July 2020
  • Change notice - Changes to external app install links effective 10 January 2020
  • Security Advisory - Changes to how apps are installed by URL in Bitbucket Cloud effective 14 October 2019
  • Change notice - Updates to Bitbucket Cloud repository URL paths
  • Deprecation notice - major changes to Bitbucket Cloud REST APIs to improve user privacy effective 12 October 2018
  • Deprecation notice - Bitbucket Cloud REST API version 1 is deprecated effective 30 June 2018
  • Announcement - New 2.0 APIs recently released!
  • Migration Guide - How to adapt to the latest REST API changes regarding privacy
